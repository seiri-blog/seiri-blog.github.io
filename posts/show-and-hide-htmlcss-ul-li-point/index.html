本記事は ul li の点を表示・非表示する方法を紹介します。
具体的には下記を参照していただければと思います。

ul li の点を非表示する方法#

<ul>
  <li>内容1</li>
  <li>内容2</li>
</ul>

ul {
  list-style: none;
}
  • 実際の結果

  • 左側の Padding があるため、必要に応じて padding-left: 0;を指定する必要があります。
<ul>
  <li>内容1</li>
  <li>内容2</li>
</ul>

ul {
  list-style: none;
  padding-left: 0;
}
  • 実際の結果

ul li の点を表示する方法#

ul li の点を ● に表示する方法#

<ul>
  <li>内容1</li>
  <li>内容2</li>
</ul>

ul {
  list-style: disc;
}
  • 実際の結果

ul li の点を ○ に表示する方法#

<ul>
  <li>内容1</li>
  <li>内容2</li>
</ul>

ul {
  list-style: circle;
}
  • 実際の結果
